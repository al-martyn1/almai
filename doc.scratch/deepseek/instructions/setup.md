Чтобы установить пакет `almai-md-split` из исходников (предоставленного проекта), выполните следующие шаги:

### 1. Подготовка
Убедитесь, что у вас установлен Python версии 3.7 или выше и pip.

### 2. Установка из корневой директории проекта
Перейдите в каталог, где находится файл `pyproject.toml` (корень проекта), и выполните:
```bash
pip install .
```
Эта команда:
- установит все зависимости (`platformdirs`, `unidecode`),
- создаст исполняемый скрипт `almai-md-split` (в виртуальном окружении или глобально),
- скопирует файл словаря по умолчанию (`conf/almai-md-split.dict`) в соответствующее место (определяемое `platformdirs`).

### 3. Проверка
После установки можно запустить:
```bash
almai-md-split --help
```
Вы должны увидеть справку и информацию о словарях.

### 4. Установка в режиме разработки (editable)
Если вы планируете изменять код, удобно установить в режиме "редактирования":
```bash
pip install -e .
```
Тогда изменения в исходниках будут сразу доступны без переустановки.

### 5. Установка из архива
Если у вас есть ZIP-архив с проектом, распакуйте его и выполните ту же команду `pip install .` внутри распакованной папки.

### 6. Создание и установка wheel
Можно собрать колесо и установить его:
```bash
pip install build   # если нет build
python -m build
pip install dist/almai_md_split-0.1.0-py3-none-any.whl
```

### Примечания
- Для работы рекомендуется использовать виртуальное окружение.
- Если `platformdirs` не установлена, скрипт выдаст предупреждение и будет использовать fallback-путь `../conf` относительно исполняемого файла. Но лучше установить её автоматически через pip.
- Файл словаря по умолчанию будет установлен в системную или пользовательскую директорию конфигов в зависимости от ОС. Вы также можете указать собственный словарь через опцию `--dict`.

После установки вы можете пользоваться утилитой командой `almai-md-split` как описано в README.
