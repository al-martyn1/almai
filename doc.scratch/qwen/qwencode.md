Конечно! Qwen Code — это открытый ИИ-агент для работы с кодом, который живет прямо в вашем терминале (CLI). Он создан командой Alibaba Cloud для того, чтобы помогать программистам писать, редактировать и отлаживать код, не выходя из консоли.
Главные фишки:

* Работа в терминале: Вы можете давать команды вроде «найди баг в этом файле» или «напиши тесты для этой функции» прямо в командной строке.
* Мощные модели: Он использует модели серии Qwen2.5-Coder (или новейшие Qwen3-Coder), которые отлично понимают логику программирования и поддерживают огромный контекст (до 1 млн токенов).
* Автономность: Агент может сам просматривать структуру вашего проекта, читать файлы и предлагать правки.
* Интеграция: Помимо терминала, его можно подключить к VS Code, JetBrains и другим редакторам через расширения.

Как начать пользоваться:
Для работы обычно требуется Node.js (версии 20+).

   1. Установка:
   
   npm install -g @qwenlm/qwen-code
   
   2. Запуск:
   Просто введите qwen в папке с вашим проектом.
   3. Настройка:
   Вам понадобится API-ключ (например, от Alibaba Cloud DashScope или платформ вроде Together AI), который вводится через команду /auth внутри интерфейса.

Для чего он полезен:

* Быстрый рефакторинг: «Перепиши этот компонент на хуки».
* Поиск ошибок: «Почему этот запрос падает с 500 ошибкой?».
* Документация: «Добавь JSDoc для всех функций в папке /src».

> Да, меня интересует инструкция по установке, а также инструкция, как подключить его к VSCode

Для установки Qwen Code и его подключения к Visual Studio Code следуйте этой пошаговой инструкции.
1. Установка Qwen Code (CLI)
Для работы инструмента в терминале вам понадобится среда Node.js (версия 20.0 или выше). [1, 2] 

   1. Проверьте Node.js: Откройте терминал и введите node -v. Если версия ниже 20 или команда не найдена, установите актуальную версию с официального сайта Node.js.
   2. Установите пакет: Выполните команду для глобальной установки:
   
   npm install -g @qwen-code/qwen-code@latest
   
   3. Первый запуск: Введите qwen в терминале. При первом запуске программа предложит выбрать тему оформления.
   4. Авторизация: Внутри интерфейса Qwen введите команду /auth. У вас будет два основных варианта:
   * Qwen OAuth (Рекомендуется): Самый простой способ — вход через браузер с аккаунтом qwen.ai (дает около 1000 бесплатных запросов в день).
      * API-KEY: Если у вас есть ключ от Alibaba Cloud (Bailian/Model Studio) или другого провайдера (например, OpenRouter), выберите этот пункт и вставьте свой ключ. [1, 2, 3, 4, 5, 6, 7, 8] 
   
------------------------------
2. Подключение к Visual Studio Code
Есть два способа использовать возможности Qwen внутри VS Code: через официальный компаньон или через сторонние расширения-агенты.
Способ А: Официальное расширение Qwen Code Companion
Это расширение связывает терминальный агент с интерфейсом редактора. [9, 10] 

   1. Установка: Откройте VS Code, перейдите в раздел Extensions (Ctrl+Shift+X) и введите в поиске "Qwen Code Companion" (издатель QwenLM).
   2. Настройка:
   * Если вы уже авторизовались в CLI (шаг 1), расширение подхватит настройки автоматически.
      * Если нет, откройте настройки расширения и укажите путь к вашему API-ключу в файле settings.json (обычно находится в ~/.qwen/settings.json).
   3. Использование: Нажмите на иконку Qwen в правом верхнем углу или вызовите панель чата через Command Palette (Ctrl+Shift+P → Qwen). [9, 10, 11, 12] 

Способ Б: Через расширение Continue (для продвинутых)
Если вы предпочитаете использовать Qwen как одну из моделей в популярном расширении Continue.dev: [13] 

   1. Установите расширение Continue в VS Code.
   2. Откройте файл конфигурации config.json (через значок шестеренки в панели Continue).
   3. Добавьте модель Qwen, указав provider: openai-compatible, model: qwen3-coder-plus (или другую версию) и ваш API-ключ. [13, 14] 

------------------------------
Основные команды в терминале
После установки вы можете использовать следующие команды прямо в чате qwen:

* /help — список всех доступных команд.
* /model — выбор конкретной модели (например, переключение на более мощную Max или быструю Plus).
* /compact — сжать контекст беседы, если он стал слишком длинным. [5]
